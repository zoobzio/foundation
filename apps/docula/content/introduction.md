# Introduction

Welcome to the **Foundation** design system documentation.

## What is Foundation?

Foundation is a comprehensive design system built on top of Vue 3 and Nuxt 3. It provides:

- A powerful token-based theming system
- Reusable UI components
- Prose components for content rendering
- Layout primitives

## Getting Started

To get started with Foundation, check out the installation guide next.

## Features

### Token System

The token system allows you to define design tokens at multiple levels:

1. **Reference tokens** - Raw values like colors, spacing, etc.
2. **Semantic tokens** - Contextual meanings like `sys-primary`
3. **Element tokens** - Component-specific styling

### Components

Foundation provides a rich set of components built on Reka UI primitives.
